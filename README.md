## Korte samenvatting van wat ik heb geleerd
- Als je elif in je if-statement hebt, en je hebt bijvoorbeeld 3 x elif, dan zal alleen de eerste elif uitgevoerd worden als dat waar is en de rest wordt dan niet meer gecontroleerd.


- ```i``` staat dus voor het aantal in ```range```. Dus als ```range(1,11)``` is zal ```i``` 1 t/m 10 keer geprint worden.


- Je kan een conditie tussen curly brackets plaatsen om te kijken of het True of False is. 
  - Bijvoorbeeld ``print(f'{john} is gelijk aan {john} : {john == john}')``
